## 🚢 Titanic Disaster Dashboard (Power BI)

This project presents an interactive data visualization of passenger data from the Titanic Disaster (1912), built using Power BI. The dashboard explores key factors that influenced survival, including gender, passenger class, age, fare, and embarkation port.

---

## 📊 Key Insights & Features

- **Total Passengers:** 891  
- **Survived Passengers:** 342  
- **Survival Rate:** ~38%  
- **Gender Distribution:** Majority of passengers were male, while females had a higher survival rate  
- **Passenger Class:** Most passengers traveled in 3rd class  
- **Port of Embarkation:** Most passengers boarded from Southampton  

### 📌 Average Metrics
- **Average Age:** 30 years  
- **Average Ticket Fare:** £32.20  

---

## 📈 Visualizations Included

- Passenger distribution by embarkation port  
- Passenger count by class  
- Gender distribution of passengers  
- Survival analysis by gender  
- Overall survival rate gauge  

---

## 🎯 Objective

The goal of this project is to analyze and visualize the Titanic dataset to uncover patterns and trends, highlighting how socio-economic and demographic factors influenced survival outcomes.

---

## 🛠️ Tools Used

- **Power BI** – Data visualization and dashboard creation  
- **Excel / CSV Dataset** – Data source  
- **Power Query** – Data cleaning and transformation
