# Titanic-Survival-Analysis
This project explores passenger data from the Titanic disaster using Power BI to uncover patterns in survival, demographics, and ticket classes.

Dashboard Overview
The dashboard provides interactive insights into:
1.Total number of passengers
2.Survival distribution
3.Average age of passengers
4.Average fare paid
5.Passenger distribution by embarkation port
6.Class distribution (Pclass)
7.Family presence (siblings/spouses, parents/children)

Key Visualizations;
1.Passenger Count by Embarkation Port
→ Shows where most passengers boarded from
2.Embarkation Port by Passenger Class (Pclass)
→ Highlights relationship between class and boarding location
3.Survival Count by Gender
→ Reveals survival differences between males and females
4.Number of Parents/Children by Gender
→ Shows family structure across genders
5.Average Fare Paid
→ Helps understand economic differences among passengers
6.Total Passenger Count KPI
→ Overview of dataset size

Key Insights;
1.Female passengers had a significantly higher survival rate than males
2.First-class passengers had better survival chances
3.Higher fares were generally associated with higher survival rates
4.Most passengers embarked from Southampton

Tools Used;
Power BI
Microsoft Excel (Data Cleaning & Preparation)
